PANDUAN PASANG NEUTRAL MODERN ACTION PALETTE

Perubahan:
1. Button/action dengan gradasi merah-biru dipaksa menjadi warna solid netral.
2. Tombol Edit/Detail/Ubah dibuat neutral graphite:
   - background #374151
   - hover #1F2937
   - teks/icon putih
3. Tombol utama dibuat slate modern:
   - background #334155
   - hover #1F2937
   - teks/icon putih
4. Tombol hapus/tolak tetap danger tetapi lebih muted:
   - background #7F1D1D
   - hover #5F1717
5. Tombol setujui/approve dibuat muted green:
   - background #166534
6. Button terang/outline memakai teks gelap.
7. Navbar active dibuat netral terang dengan teks gelap.
8. Disabled state dibuat tetap terbaca.
9. Tidak mengubah route, API, logic KDE, logic peta, atau struktur data.

Cara pasang:
1. Backup folder resources lama.
2. Extract ZIP ini.
3. Copy folder resources ke root project Laravel.
4. Replace/overwrite folder resources lama.
5. Jalankan:
   php artisan optimize:clear
   php artisan view:clear
   npm run dev
6. Hard refresh browser:
   Ctrl + Shift + R
