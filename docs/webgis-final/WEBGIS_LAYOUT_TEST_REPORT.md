# WebGIS Layout Test Report

Pengujian dilakukan terhadap hasil build produksi pada tiga ukuran viewport.

## 390 × 844 — Mobile

- Map shell: 358 × 720 px.
- Kontrol kiri: 42 × 212 px.
- Pencarian: 282 × 42 px.
- Stack panel kanan: 284 × 622 px.
- Tidak ada irisan bounding box antara kontrol kiri, pencarian, dan panel kanan.
- Panel Keterangan Peta dibuka dan tetap berada dalam stack kanan.

## 768 × 900 — Tablet

- Map shell: 720 × 684 px.
- Kontrol kiri: 44 × 222 px.
- Pencarian: 638 × 44 px.
- Stack panel kanan: 250 × 576 px.
- Tidak ada irisan bounding box antarzona kontrol.

## 1440 × 1000 — Desktop

- Map shell: 1376 × 830 px.
- Kontrol kiri: 44 × 222 px.
- Pencarian: 390 × 44 px.
- Stack panel kanan: 292 × 776 px.
- Tidak ada irisan bounding box antarzona kontrol.
